Active Window Follower










Usage:

First, use "Start Capture" to create a list of windows that are in focus. When you switch to a window/app
it will be added to the list. Click "Stop Capture" to fill the lists of windows/apps. Next, select what
window/app each menu will follow.

Make sure to select the serial port that FreeTouchDeck is on.

When done click save.
